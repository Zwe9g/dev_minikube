Dienste

AWS

Für die AWS Cloud, gibt es folgende Dienste:
Storage = Wie zum Beispiel: S3 (Simple Storage Service), Glacie
Databases = Wie zum Beispiel: DynamoDB
Compute = Amazon EC (Elastic Compute Cloud)
Serverless Compute = AWS Lambda


AZURE

Für die AZURE Cloud, gibt es folgende Dienste:
Storage = Wie zum Beispiel: Azure Blob Storage
Databases = Wie zum Beispiel: AZURE SQL Databases
Compute = Azure Virtual Machines
Serverless Compute = AZURE Functions Dienste = Intune (Verwaltung von technischen Geräten wie zum Beispiel Handys) oder auch Onedrive


Linode

Für die Linode Cloud, gibt es folgende Dienste:
Storage = Wie zum Beispiel: Linode Object Storage
Databases = Wie zum Beispiel: Linode Managed Databases
Orchistierung = Kubernetes



Git Befehle und Erklährungen

Git Befehl Erklärung
git init Initialisiert ein neues Git-Repository
git clone <repository_url> Klont ein vorhandenes Git-Repositorys
git add <file> "oder auch" git add . ; Fügt eine Datei oder das ganze Repo zur Staging-Area hinzu
git commit -m "Commit message" Erstellt einen Commit mit einer Bedeutung in der Staging-Area
git push Lädt lokale Commits in ein entferntes Repository hoch
git pull Lädt die neuesten Änderungen aus einem Remote-Repository herunter
git branch Zeigt eine Liste der vorhandenen Branches an
git checkout <branch> Wechselt zu einem anderen Branch im Repository
git merge <branch> Führt die Änderungen aus einem anderen Branch zusammen
git status Zeigt den aktuellen Status des Repositorys an
